== Description ==

With this plugin you can create menus for administration and assign roles and then display them in the frontend of the website. This plugin greatly streamlines workflow managers 

**BASIC OPTIONS**  

* **MANAGE MENU**  
Creates a menu for role

* **CUSTOMIZE YOUR MENU**  
Being wordpress menus , you can create as many of the menu items as you wish and manage your way


== Installation ==

This section describes how to install the plugin and get it working.

1. Uncompress the download package
2. Upload folder including all files the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
